## 技能說明
- **核心描述**：進行關聯式資料庫的架構規劃、資料庫 Schema 設計與性能調優，擅長與後端應用無縫整合，並進行大規模異質多源數據的 ETL 整理、存取優化與持久化管理。
- **熟練程度**：熟練
- **核心技術**：PostgreSQL, SQL Server, Schema 設計, 查詢調優, ETL 自動化

## 實際專案紀錄

### 後端關聯式資料庫整合
- **情境 (Situation)**：系統需要高度可靠、一致且支援複雜查詢的資料儲存方案，以支援各項業務邏輯及 API 端點。
- **任務 (Task)**：規劃、設計並整合關聯式資料庫以對接後端系統。
- **行動 (Action)**：採用 PostgreSQL 作為核心資料庫，進行資料庫 Schema 的規劃與關聯性設計，確保存取資料的完整性與效率。
- **結果 (Result)**：建立結構完整、效能穩定的資料持久化層，順暢支援高頻率的資料庫請求。

### 多來源資料自動化 ETL 整合服務
- **情境 (Situation)**：企業專案中有多個破碎且格式不一的外界資料來源（Fragmented sources），造成後端進行即時數據分析、視覺化和報表呈現上的極大障礙。
- **任務 (Task)**：設計並實現自動化 ETL（擷取、轉換、載入）管道，以統一清理、轉換並整併這些零散資料。
- **行動 (Action)**：設計並建立自動化 ETL 流程，將碎片化資料彙整、清洗後穩定載入 PostgreSQL 或 SQL Server 資料庫中，同時配合後端開發出專屬的異質資料格式轉換服務，為即時儀表板提供高效的數據底層。
- **結果 (Result)**：成功實現多維度零散資料的自動化整合，提供前端高度一致的即時資料視覺化及儀表板報表，大幅縮短了手動彙整的時間，提升了數據決策效率。
