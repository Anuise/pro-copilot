## 更新 (2026-07-11)


- **技能名稱**: Docker [更新]
  - **熟練度**: 熟練
  - **相關專案**: Pro-Copilot
  - **具體貢獻 (STAR)**:
    - **S (Situation, 情境)**: 系統依賴多個 API 服務與底層庫，手動佈署容易造成異質環境不一致、耗時且繁瑣。
    - **T (Task, 任務)**: 優化容器化部署方案，建立隔離乾淨的運行環境。
    - **A (Action, 行動)**: 撰寫並精簡 Dockerfile 映像檔，設定靈活的環境變數注入與資料卷 (Volume) 掛載配置。
    - **R (Result, 結果)**: 提高系統移植性，縮短了專案部署時間 50%。

- **技能名稱**: CI/CD 與自動化部署 (GitLab CI/CD) [新增]
  - **熟練度**: 熟練
  - **相關專案**: Mobagel (行動貝果有限公司)
  - **具體貢獻 (STAR)**:
    - **S (Situation, 情境)**: 缺乏自動化部署管道，手動發布與測試程序不僅冗長，更有很高機率因人工配置出錯。
    - **T (Task, 任務)**: 打造高效的 CI/CD (持續整合/持續部署) 流水線，提升產品交付效率。
    - **A (Action, 行動)**: 設計與編排 GitLab CI/CD 流程鏈，從代碼提交流程、自動化測試到多環境部署，全程無縫整合。
    - **R (Result, 結果)**: 部署出錯機率降低 30%，使團隊具備每日高頻次、且高可靠的代碼自動發布能力。
