## 技能說明
- **核心描述**：使用 React 及其上層框架 Next.js 進行現代化、響應式前端網頁架構的設計與開發，擅長優化頁面渲染效率（SSR/SSG）、加速載入速度並打造高度互動、反應敏捷的使用者動態介面。
- **熟練程度**：精通 / 熟練
- **核心技術**：React, Next.js, 動態使用者介面, 網頁效能優化, API 串接

## 實際專案紀錄

### 前端系統介面重構與優化
- **情境 (Situation)**：既有的前端介面效能與載入速度不盡理想，影響終端使用者之操作流暢度。
- **任務 (Task)**：重新規劃、設計與重構前端使用者介面，以極大化頁面效能與降低載入延遲。
- **行動 (Action)**：導入 Next.js 框架與 React 庫，重構整體前端系統，運用 Next.js 的路由與渲染優勢（如 SSR/SSG 技術），優化程式碼包裝與資源載入規則。
- **結果 (Result)**：顯著縮短頁面渲染時間，將前端載入速度提升達 40%，有效提升使用者體驗指標（Page Experience）。

### 企業專案動態前端網頁與介面開發
- **情境 (Situation)**：多個生產級網頁應用程式需要高度互動性且反應敏捷的使用者操作介面，以實時展示複雜的業務邏輯或多維度數據指標。
- **任務 (Task)**：利用前端技術開發高響應性、動態的使用者介面，提供優秀的終端用戶體驗。
- **行動 (Action)**：使用 React 框架，依據系統功能特性建立邏輯高度解耦的動態組件、規劃完備的狀態管理與頁面渲染機制，並高效串接後端多通路 API 以呈現流暢交互。
- **結果 (Result)**：極大提升了網頁應用程式的端到端（End-to-End）實時互動體驗，獲得高質感的系統視覺界面與順暢的工作流程。
