- **技能名稱**: 自動化 ETL 與資料集成
  - **熟練度**: 熟練
  - **相關專案**: Mobagel (行動貝果有限公司)
  - **具體貢獻 (STAR)**:
    - **S (Situation, 情境)**: 系統所需之外部源數據極度零散、異質平台數據規範混雜，導致難以即時在前端進行商業分析。
    - **T (Task, 任務)**: 構建高可靠性的自動化數據管道，將零散數據集中化存儲。
    - **A (Action, 行動)**: 設計、實現自動化 ETL Pipelines，將破碎的外部資料清洗與整合，集中匯入至 SQL Server 或 PostgreSQL 數據庫中，並開發對應微服務處理各種非標準格式的外部數據。
    - **R (Result, 結果)**: 提供穩定的高質量數據流，順利支援前端儀表板的實時觀測與數據視覺化，排除手動彙整的沉重負擔。
