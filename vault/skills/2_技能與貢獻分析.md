### ## AI應用與 RAG [更新]

- **技能名稱**: LLM 與 RAG 實作
  - **熟練度**: 熟練
  - **相關專案**: Mobagel (行動貝果有限公司)
  - **具體貢獻 (STAR)**:
    - **S (Situation, 情境)**: 企業內部沉澱大量非結構化、碎片化的智慧文檔，人工手動查找文檔費時且效率低落。
    - **T (Task, 任務)**: 需導入自動化檢索系統，在縮短手動耗時的同時，提升資訊搜尋的精準度。
    - **A (Action, 行動)**: 實作大語言模型 (LLM) 與檢索增強生成 (RAG) 架構，建置並優化文檔檢索 pipeline 的切分、索引與模型調校。
    - **R (Result, 結果)**: 大幅縮減人工檢索處理時間，大幅優化了文件回覆的精確度與資訊可靠性。

### ## 數據工程 [新增]

- **技能名稱**: 自動化 ETL 與資料集成
  - **熟練度**: 熟練
  - **相關專案**: Mobagel (行動貝果有限公司)
  - **具體貢獻 (STAR)**:
    - **S (Situation, 情境)**: 系統所需之外部源數據極度零散、異質平台數據規範混雜，導致難以即時在前端進行商業分析。
    - **T (Task, 任務)**: 構建高可靠性的自動化數據管道，將零散數據集中化存儲。
    - **A (Action, 行動)**: 設計、實現自動化 ETL Pipelines，將破碎的外部資料清洗與整合，集中匯入至 SQL Server 或 PostgreSQL 數據庫中，並開發對應微服務處理各種非標準格式的外部數據。
    - **R (Result, 結果)**: 提供穩定的高質量數據流，順利支援前端儀表板的實時觀測與數據視覺化，排除手動彙整的沉重負擔。

### ## 系統部署 [更新]

- **技能名稱**: Docker 容器化技術
  - **熟練度**: 熟練
  - **相關專案**: Pro-Copilot
  - **具體貢獻 (STAR)**:
    - **S (Situation, 情境)**: 系統依賴多個 API 服務與底層庫，手動佈署容易造成異質環境不一致、耗時且繁瑣。
    - **T (Task, 任務)**: 優化容器化部署方案，建立隔離乾淨的運行環境。
    - **A (Action, 行動)**: 撰寫並精簡 Dockerfile 映像檔，設定靈活的環境變數注入與資料卷 (Volume) 掛載配置。
    - **R (Result, 結果)**: 提高系統移植性，縮短了專案部署時間 50%。

- **技能名稱**: GitLab CI/CD 與自動化部署
  - **熟練度**: 熟練
  - **相關專案**: Mobagel (行動貝果有限公司)
  - **具體貢獻 (STAR)**:
    - **S (Situation, 情境)**: 缺乏自動化部署管道，手動發布與測試程序不僅冗長，更有很高機率因人工配置出錯。
    - **T (Task, 任務)**: 打造高效的 CI/CD (持續整合/持續部署) 流水線，提升產品交付效率。
    - **A (Action, 行動)**: 設計與編排 GitLab CI/CD 流程鏈，從代碼提交流程、自動化測試到多環境部署，全程無縫整合。
    - **R (Result, 結果)**: 部署出錯機率降低 30%，使團隊具備每日高頻次、且高可靠的代碼自動發布能力。
